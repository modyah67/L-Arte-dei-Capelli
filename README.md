# L’Arte dei Capelli — Hair Salon Website (Exam Project) 💇‍♂️

This is a multi-page static website created as a final exam project during my web development course. It focuses on semantic HTML5, modular CSS, and responsive design.

## 🌟 Key Features
- **Responsive Navigation:** Multi-level mobile menu (burger menu) implemented for different screen sizes.
- **Modern Layout:** Extensive use of CSS Grid for the portfolio gallery and Flexbox for component alignment.
- **Interactive Elements:** Fixed background effects (parallax style), contact forms, and integrated Google Maps.
- **Performance:** Usage of modern image formats (.avif) and lazy loading for better speed.

## 🛠 Tech Stack
- HTML5 (Semantic elements)
- CSS3 (Custom variables, Grid, Flexbox, Animations)
- FontAwesome Icons
- Google Fonts Integration

## 📌 Project Context
This was an examination project to demonstrate my skills in converting design layouts into functional code. While it is a static site, it serves as a foundation for my journey into front-end development.
